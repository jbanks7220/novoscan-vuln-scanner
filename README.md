# Web Vulnerability Scanner (Portfolio Project)

⚠️ **Ethical Use Notice**
This tool is intended for educational purposes and for scanning systems you own or have explicit permission to test.
Unauthorized scanning of external systems is illegal.

## Features
- GUI-based vulnerability scanner
- Localhost / authorized target scanning
- Open port detection
- Simulated vulnerability checks
- Threat rating (Low / Medium / High / Critical)
- Export reports in:
  - Markdown (.md)
  - HTML (.html)
  - JSON (.json)
  - XML (.xml)

## How to Run
```bash
pip install -r requirements.txt
python main.py
